## Projeto

Oum sistema desenvolvido em Python para simular o monitoramento de uma missão espacial., o programa analisa temperatura, comunicação, bateria, oxigênio e estabilidade, gerando alertas e um relatório final.

## Objetivo
O projeto mostra como a programação pode ser usada para monitorar sistemas críticos e auxiliar na tomada de decisões por meio da análise de dados.
